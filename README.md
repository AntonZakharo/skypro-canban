# Доска-канбан

## Проект создан по курсу skypro для обучения основам Vue. \n Это веб-сайт, представляющий собой доску-канбан, в которой реализованы функции добавления, удаления, редактирования заданий, связь с API, авторизация и регистрация и смена на темную тему.
## Использованные технологии: Vue 3.5, VueRouter, VueUse JavaScript, HTML, CSS, SCSS.
